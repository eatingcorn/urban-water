# Analyzing the Impact of Mexico’s Clean Water Program, Programa Agua Limpia, on Child Mortality Rates∗

## Overview

This GitHub Repository contains the Analysis of the Impact of Mexico’s Clean Water Program, Programa Agua Limpia, on Child Mortality Rates.

## File Structure

The repo is structured as:

-   `data/raw_data` contains the raw data as obtained from "Urban Water Disinfection and Mortality Decline in Lower-Income Countries" by Sonia R. Bhalotra, Alberto Diaz-Cayeros, Grant Miller, Alfonso Miranda, and Atheendar S. Venkataramani.
-   `data/analysis_data` contains the cleaned dataset that was constructed.
-   `other/sketches` contains relevant literature, details about LLM chat interactions, and sketches.
-   `other/ssrp_report` contains the files used to generate the ssrp_report, including the Quarto document and reference bibliography file, as well as the PDF of the report attached to SSRP reproduction [https://www.socialsciencereproduction.org/reproductions/784a7e5a-9b55-41e0-b832-02f823fe012f/index]
-   `paper` contains the files used to generate the paper, including the Quarto document and reference bibliography file, as well as the PDF of the paper. 
-   `scripts` contains the R scripts used to simulate, download, clean data, test, and replicate graphs.

## Reproduction

The replication R code for figures can be found `scripts/99-replications.R`.

## Original Data

The original data can be found and downloaded here : [https://doi.org/10.1257/pol.20180764]

## Statement on LLM usage

Aspects of the code were written with the help of ChatGPT 3.5 and the entire chat history is available in inputs/llms/usage.txt.
